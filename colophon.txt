mrzool.cc / Colophon
====================

This website is hosted on GitHub Pages.

Its HTML gets generated by Jekyll from a bunch of YAML and Markdown files.

The source code can be found here: https://github.com/mrzool/mrzool.github.io

CSS styling is provided by Gravitons (http://jxnblk.com/gravitons/).

The text is typeset in whatever your OS default sans-serif is.

No cookies, no analytics, no tracking, no bullshit.

Proudly written in Vim.
