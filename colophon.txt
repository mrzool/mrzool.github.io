mrzool.cc / Colophon
====================

This website is hosted on GitHub Pages. The static HTML gets generated by Jekyll from a bunch of YAML and Markdown files.

The source code can be found here: https://github.com/mrzool/mrzool.github.io

Styling is provided by Gravitons (http://jxnblk.com/gravitons/).

The text is typeset in whatever the default sans-serif on your system is.

No cookies, no analytics, no tracking.

Thanks for visiting.
